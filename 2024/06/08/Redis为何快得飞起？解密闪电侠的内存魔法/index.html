еҚ•зәҝзЁӢжЁЎеһӢпјҡеҢ–з№Ғдёәз®Җзҡ„жһҒиҮҙ

Redis жҳҜеҚ•зәҝзЁӢпјҢдё»иҰҒжҳҜжҢҮ Redis зҡ„зҪ‘з»ңIOе’Ңй”®еҖјеҜ№иҜ»еҶҷжҳҜз”ұдёҖдёӘзәҝзЁӢжқҘе®ҢжҲҗзҡ„пјҢиҝҷд№ҹжҳҜ Redis еҜ№еӨ–жҸҗдҫӣй”®еҖјеӯҳеӮЁжңҚеҠЎзҡ„дё»иҰҒжөҒзЁӢгҖӮ

еӨҡзәҝзЁӢеҝ…з„¶дјҡйқўдёҙеҜ№дәҺе…ұдә«иө„жәҗзҡ„и®ҝй—®пјҢиҝҷж—¶еҖҷйҖҡеёёзҡ„еҒҡжі•е°ұжҳҜеҠ й”ҒпјҢиҷҪз„¶жҳҜеӨҡзәҝзЁӢпјҢиҝҷж—¶еҖҷе°ұдјҡеҸҳжҲҗдёІиЎҢзҡ„и®ҝй—®гҖӮд№ҹе°ұжҳҜеӨҡзәҝзЁӢзј–зЁӢжЁЎејҸдјҡйқўдёҙзҡ„е…ұдә«иө„жәҗзҡ„е№¶еҸ‘и®ҝй—®жҺ§еҲ¶й—®йўҳгҖӮ

еҗҢж—¶еӨҡзәҝзЁӢд№ҹдјҡеј•е…ҘеҗҢжӯҘеҺҹиҜӯжқҘдҝқжҠӨе…ұдә«иө„жәҗзҡ„е№¶еҸ‘и®ҝй—®пјҢд»Јз Ғзҡ„еҸҜз»ҙжҠӨжҖ§е’Ңжҳ“иҜ»жҖ§е°ҶдјҡдёӢйҷҚгҖӮ

Redis зҡ„зәҝзЁӢжЁЎеһӢеңЁдёҚеҗҢзүҲжң¬жңүжүҖдёҚеҗҢпјҡ

  • 2.0 зүҲжң¬пјҡRedis дҪҝз”ЁеҚ•дёӘзәҝзЁӢеңЁдәӢд»¶еҫӘзҺҜдёӯеӨ„зҗҶзҪ‘з»ңиҜ·жұӮдёҺжү§иЎҢж“ҚдҪңжҢҮд»ӨпјҢз„¶еҗҺе…¶д»–зҡ„еҗҺеҸ°зәҝзЁӢиҙҹиҙЈйҮҠж”ҫ RDB/AOF иҝҮзЁӢз”ҹжҲҗзҡ„дёҙж—¶ж–Үд»¶иө„жәҗдёҺеҲ·зӣҳпјӣ
  • 4.0 зүҲжң¬пјҡRedis ж·»еҠ дәҶдёҖдёӘзәҝзЁӢпјҢз”ЁдәҺејӮжӯҘжү§иЎҢUNLINK(ејӮжӯҘеҲ йҷӨжҢҮе®ҡй”®)гҖҒFLUSHALL ASYNC(жё…з©әжүҖжңү DB)е’ҢFLUSHDB ASYNC(жё…з©әжҢҮе®ҡ DB)иҝҷдәӣжҜ”иҫғйҮҚзҡ„еҲ йҷӨжҢҮд»Өпјӣ
  • 6.0 зүҲжң¬пјҡRedis е…Ғи®ёйҖҡиҝҮдҝ®ж”№ io-threads е’Ң io-threads-do-reads дҝ®ж”№ IO зәҝзЁӢж•°гҖӮ

еҸҰеӨ–дёҖжҸҗпјҢиҜ»еҶҷжҢҮд»ӨиҰҒдҝқжҢҒеҚ•зәҝзЁӢпјҢиҝҷдёӘи®ҫи®Ўзҡ„зҗҶз”ұжҳҜеӣ дёә CPU еҜ№еҶ…еӯҳзҡ„ж“ҚдҪңе·Із»Ҹи¶іеӨҹй«ҳж•ҲпјҢеӣ жӯӨжҖ§иғҪз“¶йўҲдёҚеӨ§еҸҜиғҪжқҘиҮӘдәҺ CPU пјҢиҖҢдё»иҰҒжқҘиҮӘдәҺеҶ…еӯҳе’ҢзҪ‘з»ң IOпјҢеӣ жӯӨжү§иЎҢе‘Ҫд»Өзҡ„зәҝзЁӢжңүдёҖдёӘи¶ізҹЈгҖӮ

йқһйҳ»еЎһI/Oпјҡй«ҳе№¶еҸ‘еӨ„зҗҶзҡ„жі•е®қ

йҰ–е…ҲпјҢRedis жҳҜи·‘еңЁеҚ•зәҝзЁӢдёӯзҡ„пјҢжүҖжңүзҡ„ж“ҚдҪңйғҪжҳҜжҢүз…§йЎәеәҸзәҝжҖ§жү§иЎҢзҡ„пјҢдҪҶжҳҜз”ұдәҺиҜ»еҶҷж“ҚдҪңзӯүеҫ…з”ЁжҲ·иҫ“е…ҘжҲ–иҫ“еҮәйғҪжҳҜйҳ»еЎһзҡ„пјҢжүҖд»Ҙ I/O ж“ҚдҪңеңЁдёҖиҲ¬жғ…еҶөдёӢеҫҖеҫҖдёҚиғҪзӣҙжҺҘиҝ”еӣһпјҢиҝҷдјҡеҜјиҮҙжҹҗдёҖж–Үд»¶зҡ„ I/O йҳ»еЎһеҜјиҮҙж•ҙдёӘиҝӣзЁӢж— жі•еҜ№е…¶е®ғе®ўжҲ·жҸҗдҫӣжңҚеҠЎпјҢиҖҢ I/O еӨҡи·ҜеӨҚз”Ё е°ұжҳҜдёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳиҖҢеҮәзҺ°зҡ„гҖӮ

Linux дёӯзҡ„IOеӨҡи·ҜеӨҚз”ЁжңәеҲ¶жҳҜжҢҮдёҖдёӘзәҝзЁӢеӨ„зҗҶеӨҡдёӘIOжөҒгҖӮеӨҡи·ҜжҳҜжҢҮзҪ‘з»ңиҝһжҺҘпјҢеӨҚз”ЁжҢҮзҡ„жҳҜеҗҢдёҖдёӘзәҝзЁӢгҖӮз®ҖеҚ•жқҘиҜҙпјҢеңЁ Redis еҸӘиҝҗиЎҢеҚ•зәҝзЁӢзҡ„жғ…еҶөдёӢпјҢиҜҘжңәеҲ¶е…Ғи®ёеҶ…ж ёдёӯпјҢеҗҢж—¶еӯҳеңЁеӨҡдёӘзӣ‘еҗ¬еҘ—жҺҘеӯ—е’Ңе·ІиҝһжҺҘеҘ—жҺҘеӯ—гҖӮеҶ…ж ёдјҡдёҖзӣҙзӣ‘еҗ¬иҝҷдәӣеҘ—жҺҘеӯ—дёҠзҡ„иҝһжҺҘиҜ·жұӮжҲ–ж•°жҚ®иҜ·жұӮгҖӮдёҖж—ҰжңүиҜ·жұӮеҲ°иҫҫпјҢе°ұдјҡдәӨз»ҷ Redis зәҝзЁӢеӨ„зҗҶпјҢиҝҷе°ұе®һзҺ°дәҶдёҖдёӘ Redis зәҝзЁӢеӨ„зҗҶеӨҡдёӘIOжөҒзҡ„ж•ҲжһңгҖӮ

ж–Үд»¶дәӢд»¶жҳҜеҜ№иҝһжҺҘ socket ж“ҚдҪңзҡ„дёҖдёӘжҠҪиұЎгҖӮеҪ“з«ҜеҸЈзӣ‘еҗ¬ socket еҮҶеӨҮ accept ж–°иҝһжҺҘпјҢжҲ–иҖ…иҝһжҺҘ socket еҮҶеӨҮеҘҪиҜ»еҸ–иҜ·жұӮгҖҒеҶҷе…Ҙе“Қеә”гҖҒе…ій—ӯж—¶пјҢе°ұдјҡдә§з”ҹдёҖдёӘж–Үд»¶дәӢд»¶гҖӮIO еӨҡи·ҜеӨҚз”ЁзЁӢеәҸиҙҹиҙЈеҗҢж—¶зӣ‘еҗ¬еӨҡдёӘ socketпјҢеҪ“иҝҷдәӣ socket дә§з”ҹж–Үд»¶дәӢд»¶ж—¶пјҢе°ұдјҡи§ҰеҸ‘дәӢд»¶йҖҡзҹҘпјҢж–Үд»¶еҲҶжҙҫеҷЁе°ұдјҡж„ҹзҹҘе№¶иҺ·еҸ–еҲ°иҝҷдәӣдәӢд»¶гҖӮ

иҷҪз„¶еӨҡдёӘж–Үд»¶дәӢд»¶еҸҜиғҪдјҡе№¶еҸ‘еҮәзҺ°пјҢдҪҶ IO еӨҡи·ҜеӨҚз”ЁзЁӢеәҸжҖ»дјҡе°ҶжүҖжңүдә§з”ҹдәӢд»¶зҡ„ socket ж”ҫе…ҘдёҖдёӘйҳҹеҲ—дёӯпјҢйҖҡиҝҮиҝҷдёӘйҳҹеҲ—пјҢжңүеәҸзҡ„жҠҠиҝҷдәӣж–Үд»¶дәӢд»¶йҖҡзҹҘз»ҷж–Үд»¶еҲҶжҙҫеҷЁгҖӮ

ж–Үд»¶дәӢд»¶еҲҶжҙҫеҷЁжҺҘж”¶ I/O еӨҡи·ҜеӨҚз”ЁзЁӢеәҸдј жқҘзҡ„еҘ—жҺҘеӯ—пјҢе№¶ж №жҚ®еҘ—жҺҘеӯ—дә§з”ҹзҡ„дәӢд»¶зұ»еһӢпјҢи°ғз”Ёзӣёеә”зҡ„дәӢд»¶еӨ„зҗҶеҷЁгҖӮ

жңҚеҠЎеҷЁдјҡдёәжү§иЎҢдёҚеҗҢд»»еҠЎзҡ„еҘ—жҺҘеӯ—е…іиҒ”дёҚеҗҢзҡ„дәӢд»¶еӨ„зҗҶеҷЁпјҢиҝҷдәӣеӨ„зҗҶеҷЁжҳҜдёҖдёӘдёӘзҡ„еҮҪж•°пјҢд»–д»¬е®ҡд№үдәҶиҝҷдёӘдәӢд»¶еҸ‘з”ҹж—¶пјҢжңҚеҠЎеҷЁеә”иҜҘжү§иЎҢзҡ„еҠЁдҪңгҖӮ

Redis е°ҒиЈ…дәҶ 4 з§ҚеӨҡи·ҜеӨҚз”ЁзЁӢеәҸпјҢжҜҸз§Қе°ҒиЈ…е®һзҺ°йғҪжҸҗдҫӣдәҶзӣёеҗҢзҡ„ API е®һзҺ°гҖӮзј–иҜ‘ж—¶пјҢдјҡжҢүз…§жҖ§иғҪе’Ңзі»з»ҹе№іеҸ°пјҢйҖүжӢ©жңҖдҪізҡ„ IO еӨҡи·ҜеӨҚз”ЁеҮҪж•°дҪңдёәеә•еұӮе®һзҺ°пјҢйҖүжӢ©йЎәеәҸжҳҜпјҢйҰ–е…Ҳе°қиҜ•йҖүжӢ© Solaries дёӯзҡ„ evportпјҢеҰӮжһңжІЎжңүпјҢе°ұе°қиҜ•йҖүжӢ© Linux дёӯзҡ„ epollпјҢеҗҰеҲҷе°ұйҖүжӢ©еӨ§еӨҡ UNIX зі»з»ҹйғҪж”ҜжҢҒзҡ„ kqueueпјҢиҝҷ 3 дёӘеӨҡи·ҜеӨҚз”ЁеҮҪж•°йғҪзӣҙжҺҘдҪҝз”Ёзі»з»ҹеҶ…ж ёеҶ…йғЁзҡ„з»“жһ„пјҢеҸҜд»ҘжңҚеҠЎж•°еҚҒдёҮзҡ„ж–Үд»¶жҸҸиҝ°з¬ҰгҖӮ

еҰӮжһңеҪ“еүҚзј–иҜ‘зҺҜеўғжІЎжңүдёҠиҝ°еҮҪж•°пјҢе°ұдјҡйҖүжӢ© select дҪңдёәеә•еұӮе®һзҺ°ж–№жЎҲгҖӮselect ж–№жЎҲзҡ„жҖ§иғҪиҫғе·®пјҢдәӢд»¶еҸ‘з”ҹж—¶пјҢдјҡжү«жҸҸе…ЁйғЁзӣ‘еҗ¬зҡ„жҸҸиҝ°з¬ҰпјҢдәӢд»¶еӨҚжқӮеәҰжҳҜ O(n)пјҢе№¶дё”еҸӘиғҪеҗҢж—¶жңҚеҠЎжңүйҷҗдёӘж–Үд»¶жҸҸиҝ°з¬ҰпјҢ32 дҪҚжңәй»ҳи®ӨжҳҜ 1024 дёӘпјҢ64 дҪҚжңәй»ҳи®ӨжҳҜ 2048 дёӘпјҢжүҖд»ҘдёҖиҲ¬жғ…еҶөдёӢпјҢе№¶дёҚдјҡйҖүжӢ© select дҪңдёәзәҝдёҠиҝҗиЎҢж–№жЎҲгҖӮ

еҚ•зәҝзЁӢеӨ„зҗҶ I/O иҜ·жұӮзҡ„жҖ§иғҪз“¶йўҲ

1гҖҒеҗҺеҸ° Redis йҖҡиҝҮзӣ‘еҗ¬еӨ„зҗҶдәӢд»¶йҳҹеҲ—дёӯзҡ„ж¶ҲжҒҜжқҘйҖҡиҝҮеҚ•зәҝзЁӢеӨ„зҗҶе‘Ҫд»ӨпјҢеҰӮжһңдёҖдёӘе‘Ҫд»Өзҡ„жү§иЎҢж—¶й—ҙеҫҲд№…пјҢе°ұдјҡеҪұе“Қж•ҙдёӘ server зҡ„жҖ§иғҪпјӣ
