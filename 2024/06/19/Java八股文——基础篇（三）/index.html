Exception е’Ң Error жңүд»Җд№ҲеҢәеҲ«

еңЁ Java дёӯпјҢжүҖжңүзҡ„ејӮеёёйғҪжңүдёҖдёӘе…ұеҗҢзҡ„зҘ–е…Ҳ java.lang еҢ…дёӯзҡ„ Throwable зұ»гҖӮ Throwable жңүдёӨдёӘйҮҚиҰҒзҡ„еӯҗзұ»пјҡ

  • Exception :зЁӢеәҸжң¬иә«еҸҜд»ҘеӨ„зҗҶзҡ„ејӮеёёпјҢеҸҜд»ҘйҖҡиҝҮ catch жқҘиҝӣиЎҢжҚ•иҺ·гҖӮException еҸҲеҸҜд»ҘеҲҶдёә Checked Exception (еҸ—жЈҖжҹҘејӮеёёпјҢеҝ…йЎ»еӨ„зҗҶ) е’Ң Unchecked Exception (дёҚеҸ—жЈҖжҹҘејӮеёёпјҢеҸҜд»ҘдёҚеӨ„зҗҶ)гҖӮ
  • ErrorпјҡError еұһдәҺзЁӢеәҸж— жі•еӨ„зҗҶзҡ„й”ҷиҜҜ пјҢдёҚе»әи®®йҖҡиҝҮcatchжҚ•иҺ· гҖӮдҫӢеҰӮ Java иҷҡжӢҹжңәиҝҗиЎҢй”ҷиҜҜпјҲVirtual MachineErrorпјүгҖҒиҷҡжӢҹжңәеҶ…еӯҳдёҚеӨҹй”ҷиҜҜ(OutOfMemoryError)гҖҒзұ»е®ҡд№үй”ҷиҜҜпјҲNoClassDefFoundErrorпјүзӯү гҖӮиҝҷдәӣејӮеёёеҸ‘з”ҹж—¶пјҢJava иҷҡжӢҹжңәпјҲJVMпјүдёҖиҲ¬дјҡйҖүжӢ©зәҝзЁӢз»ҲжӯўгҖӮ
